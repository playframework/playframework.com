Play Seed Templates

Experienced With Play? Use a Seed Template!

If you already know a bit about Play and have sbt 0.13.13 or higher installed, you can create your own Play project using the sbt new command with a giter8 template. The giter8 templates are similar to Maven archetypes. We also provide a zipped version of the seed projects that you can download below.

You can also create your own giter8 seeds and scaffolds based off ours by forking from the https://github.com/playframework/play-java-seed.g8 or https://github.com/playframework/play-scala-seed.g8 github projects.

Note that the seed templates are already configured with CSRF and security headers filters, whereas the other example projects are not specifically set up for security out of the box.

Use sbt run and then go to http://localhost:9000 to see the running server.
