Play Starter Projects

New to Play? Start here!

Play starter projects include a packaged version of sbt, a build tool that works well with Play. The source code includes comments describing how the example works with links to related documentation.

To try a starter project:

  • Use the link to download the zipped project
  • Extract the zipped project.
  • In a command window, change to the top project directory.
  • Enter the following at the prompt to build and run the starter project: 
    ./sbt run
  • Enter the following URL in a browser: 
    http://localhost:9000

    A welcome page displays with information about the example app.
