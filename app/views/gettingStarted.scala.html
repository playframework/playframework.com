It's quick and easy to get started

What would you like to do?

To learn about Play hands-on, try the examples as described below. When you are ready to build your own app, use a seed template that sets up the project structure and dev environment for you. You can easily integrate Play projects into your favorite IDE.

Downloading and building examples

Lightbend Tech Hub offers a variety of Play examples for Java and Scala. We recommend trying the Play Starter Example first.

The downloadable zip files include everything you need to build and run the examples, including a distribution of the Simple Build Tool sbt. If you have Gradle, you can use it as a build tool instead.

After choosing an example from Tech Hub follow these steps:

  1. Click CREATE A PROJECT FOR ME to download the zipped project.
  2. Unzip the project in a convenient location.
  3. In a command window, navigate to the top level project directory.
  4. Enter one of the following commands:
    • On OSx or Linux systems: ./sbt run or ./gradle runPlayBinary
    • On Windows systems: sbt.bat run or gradlew.bat runPlayBinary

    The build tool downloads dependencies and compiles the project.

  5. After the build finishes, enter the following URL in a browser to view the app:

    http://localhost:9000/

Check out the readme file in the top level project directory to learn more about the example.
